1. 38外观数列：字符串里找相同字母连续的有多少次

```java
for(int j = 0; j < len; j++){
    int k = j;
    while(k < len && s.charAt(k) == s.charAt(j)){
        k++;
    }
    // 因为循环体里还会有j++，所以这边要减去1
    j = k - 1;
}
```
