# LeetCode 专题 -- 二分查找专题

## 69. x 的平方根

`难度：简单`

### 题目描述

实现 int sqrt(int x) 函数。

计算并返回 x 的平方根，其中 x 是非负整数。

由于返回类型是整数，结果只保留整数的部分，小数部分将被舍去。

示例 1:

```matlab
输入: 4
输出: 2
```

示例 2:

```matlab
输入: 8
输出: 2
说明: 8 的平方根是 2.82842...,
     由于返回类型是整数，小数部分将被舍去。
```

**链接**：
> <https://leetcode-cn.com/problems/sqrtx/>

### Solution

```java
class Solution {
    public int mySqrt(int x) {
        // 最后一个平方小于等于x的数
        long low = 1, high = x;
        while(low <= high){
            long mid = low + (high - low) / 2;
            if(mid * mid <= x){
                if(mid == x  || (mid + 1) * (mid + 1) > x) return (int)mid;
                else low = mid + 1;
            }else{
                high = mid - 1;
            }
        }
        return 0;

    }
}
```
